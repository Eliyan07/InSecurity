# Python Инструменти

Тази директория съдържа само офлайн инструменти за разработка.

Готовото десктоп приложение не използва Python по време на работа. Детекцията,
сканирането и защитата в реално време се изпълняват в Rust, а приложението
използва ONNX модели и други ресурси вместо Python среда за изпълнение.

## За Какво Се Използва

- обучение и оценяване на ML модели
- извличане на признаци от PE файлове
- подписване и проверка на pickle и joblib артефакти на модели
- изпълнение на Python тестове за помощните инструменти

## За Какво Не Се Използва

- сканиране по време на работа
- логика на потребителския интерфейс или на сървърната част
- защита в реално време
- production зависимости на крайното приложение

## Основни Файлове

- `ml_models/model_security.py` - подписва и проверява офлайн артефакти на модели
- `training/extract_features.py` - извлича признаци от PE файлове
- `training/train_novelty.py` - обучава novelty detection модела
- `tests/test_model_security.py` - тества помощните функции за подписване на модели

## За Преглед На Проекта

Ако преглеждаш самото десктоп приложение, основният код за изпълнение е в:

- `src-tauri/`
- `src/`
- `resources/models/`

Тази Python директория е допълнителна част за разработката и поддръжката на моделите.
