# Бележки За Обучението С Python

Тази папка съдържа незадължителни скриптове за офлайн ML експерименти.

Готовото приложение не изпълнява тези скриптове. Те съществуват, за да може
хранилището да възпроизвежда или развива ML ресурсите извън Rust частта за изпълнение.

## Налични Скриптове

- `extract_features.py`  
  Извлича признаци от PE файлове и ги записва в JSON.

- `train_novelty.py`  
  Обучава детектора за аномалии от подготвени набори с признаци и записва артефакти
  на модели за по-късен export или подписване.

## Типичен Процес

1. Събери benign PE примери за обучение.
2. Стартирай `extract_features.py` върху тези примери.
3. Обучи novelty модел с `train_novelty.py`.
4. Подпиши генерираните pickle/joblib артефакти с `ml_models/model_security.py`.
5. Експортирай или обнови финалните модели за изпълнение, използвани от Rust приложението.

## Важен Контекст

- Това са само офлайн инструменти.
- Оценяването в десктоп приложението използва Rust и ONNX модели.
- Ако не преобучаваш модели, тази папка не е нужна за компилация или стартиране на приложението.
