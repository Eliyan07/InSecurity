# Данни За Обучение

Тази директория е предназначена за офлайн ML данни и междинни артефакти.

Тя не е част от изпълнението на крайното десктоп приложение. Rust/Tauri приложението
зарежда моделите за работа от `resources/models/`, а тази папка се използва при
извличане на признаци, експерименти и преобучаване.

## Текуща Структура

```text
training_data/
|- features/
|  |- benign/              Извлечени JSON файлове с benign признаци
|  `- malware/             Извлечени JSON файлове с malware признаци
`- models/                 Временни обучени модели преди export или подписване
```

## Типичен Процес

1. Извлечи признаците в `training_data/features/`.
2. Обучи или оцени модели с файловете от тази директория.
3. Подпиши pickle или joblib артефакти при нужда.
4. Експортирай финалния модел за работа към `resources/models/`.

Виж `python/training/README.md` за бележки относно Python инструментите.

## Важни Бележки

- Не добавяй сурови malware примери в Git.
- Дръж тук само извлечени признаци или временни артефакти от обучението.
- Използвай законно придобити и безопасно обработвани примери за всяка работа с набори от данни.
- Ако директорията стане голяма локално, дръж я игнорирана или я почисти преди публикуване.
